Introduction

InviteManager is a Discord bot for invite tracking, analytics, leaderboards, and server management tools. This Privacy Policy explains how we handle data.

Data We Collect

  • Discord User IDs
  • Server (Guild) IDs
  • Channel & Role IDs
  • Invite codes & statistics
  • Join/leave events
  • Server settings
  • Command usage metadata

What We DO NOT Collect

  • Passwords
  • Email addresses
  • Private messages
  • Payment information
  • Permanent message content

How We Use Data

  • Invite tracking
  • Leaderboards
  • Server analytics
  • Reward systems
  • Bot functionality

Data Storage

Data is securely stored and only kept as long as needed for bot functionality.

Sharing

We do not sell or share data except when required by law.

Contact

For support or data removal requests, contact us via the official Discord support server.
